# 校园"轻"社交与资源共享平台 - H5界面设计文档

## 1. 设计概述

### 1.1 设计理念

- **轻量简洁**：界面清爽，减少视觉负担
- **年轻活力**：符合大学生审美，色彩明快
- **易用性**：操作直观，降低学习成本
- **一致性**：统一的视觉语言和交互模式

### 1.2 设计风格

- **风格定位**：现代简约、校园风
- **色彩基调**：清新、活力、温暖
- **图标风格**：线性图标，简洁明了
- **字体选择**：系统默认字体，保证可读性

### 1.3 色彩方案

**主色调**：
- 主色（Primary）：`#4A90E2`（清新蓝）- 用于主要按钮、链接、重要信息
- 辅助色（Secondary）：`#7ED321`（活力绿）- 用于成功提示、积极操作

**功能色**：
- 警告色（Warning）：`#F5A623`（橙色）
- 错误色（Error）：`#D0021B`（红色）
- 信息色（Info）：`#50E3C2`（青色）

**中性色**：
- 深色文字：`#333333`
- 次要文字：`#666666`
- 辅助文字：`#999999`
- 边框颜色：`#E5E5E5`
- 背景色：`#F7F8FA`
- 白色：`#FFFFFF`

## 2. 页面结构设计

### 2.1 整体布局

采用 **底部Tab导航 + 顶部标题栏** 的经典布局结构

```
┌─────────────────────────────┐
│      顶部导航栏/标题栏        │
├─────────────────────────────┤
│                             │
│                             │
│        主内容区域            │
│                             │
│                             │
├─────────────────────────────┤
│      底部Tab导航栏           │
└─────────────────────────────┘
```

### 2.2 底部Tab导航

**Tab项目**（从左到右）：

1. **首页** - 图标：home
2. **资源** - 图标：grid
3. **发布** - 图标：plus-circle（突出显示）
4. **消息** - 图标：message
5. **我的** - 图标：user

**设计规范**：
- Tab高度：50px
- 图标大小：24px
- 文字大小：10px
- 选中状态：主色 + 图标填充
- 未选中状态：灰色 + 图标线性

## 5. 交互设计

### 5.1 页面切换

- Tab切换：无动画，即时切换
- 页面跳转：右滑进入，左滑返回
- 弹窗：淡入淡出 + 缩放

### 5.2 手势操作

- 下拉刷新：顶部下拉
- 上拉加载：底部上拉
- 左滑删除：消息列表、历史记录
- 长按：显示更多操作菜单

### 5.3 反馈效果

- 按钮点击：轻微缩放 + 透明度变化
- 列表项点击：背景色变化
- 加载：骨架屏或加载动画
- 成功/失败：Toast提示

### 5.4 动画效果

- 点赞：心形放大动画
- 关注：按钮文字切换动画
- 图片加载：渐显
- 列表加载：从下往上淡入

## 6. 响应式设计

### 6.1 断点设置

- 小屏：< 375px
- 中屏：375px - 414px
- 大屏：> 414px

### 6.2 适配策略

- 使用相对单位（rem、vw）
- 图片自适应
- 文字大小适当调整
- 保持关键信息可见

## 7. 无障碍设计

- 足够的点击区域（最小44px × 44px）
- 合适的颜色对比度
- 清晰的文字大小
- 语义化的HTML标签
- 支持屏幕阅读器

## 8. 性能优化

- 图片懒加载
- 列表虚拟滚动
- 骨架屏占位
- 资源预加载
- 代码分割

## 9. 设计规范总结

### 9.1 间距规范

- 最小间距：4px
- 常用间距：8px、12px、16px、24px
- 页面边距：12px

### 9.2 字体规范

- 大标题：20px
- 标题：18px
- 副标题：16px
- 正文：14px
- 辅助文字：12px
- 小字：10px

### 9.3 圆角规范

- 卡片：8px
- 按钮：4px 或 全圆角
- 头像：50%（圆形）
- 标签：全圆角

### 9.4 阴影规范

- 轻阴影：0 2px 8px rgba(0,0,0,0.08)
- 中阴影：0 4px 12px rgba(0,0,0,0.12)
- 重阴影：0 8px 16px rgba(0,0,0,0.16)
