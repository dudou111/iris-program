# 基于小程序的校园"轻"社交与资源共享平台需求文档

## 1. 项目概述

### 1.1 项目名称

校园"轻"社交与资源共享平台（Iris Program）

### 1.2 项目背景

随着移动互联网的发展，校园内学生对便捷的社交和资源共享需求日益增长。传统的校园信息传播方式效率低下，资源共享渠道分散。本平台旨在通过微信小程序，为校园用户提供轻量级的社交互动和资源共享服务。

### 1.3 项目目标

- 构建轻量级校园社交平台，降低社交门槛
- 实现校园资源高效共享与流通
- 提供便捷的信息发布与获取渠道
- 促进校园内良性互动与交流

## 2. 用户分析

### 2.1 目标用户

- 在校大学生（主要用户群体）
- 教职工
- 校园服务提供者

### 2.2 用户需求

- **社交需求**：结识同校同学、兴趣交友、活动组织
- **资源共享需求**：二手交易、学习资料、失物招领
- **信息获取需求**：校园动态、活动通知、实用信息

## 3. 功能需求

### 3.1 用户模块

#### 3.1.1 用户注册与登录
- 微信授权登录
- 学生身份认证（学号/学生证验证）
- 用户信息完善（昵称、头像、院系、年级等）

#### 3.1.2 个人中心
- 个人资料展示与编辑
- 我的发布（动态、资源）
- 我的收藏
- 我的消息
- 账号设置

### 3.2 社交模块

#### 3.2.1 动态广场
- 动态发布（文字、图片、话题标签）
- 动态浏览（瀑布流/列表展示）
- 动态互动（点赞、评论、转发）
- 话题分类（学习、生活、活动、表白墙等）
- 热门推荐

#### 3.2.2 校园圈子
- 兴趣圈子（社团、兴趣小组）
- 圈子动态
- 圈子成员管理
- 圈子活动发布

#### 3.2.3 私信功能
- 一对一聊天
- 消息通知
- 好友申请

### 3.3 资源共享模块

#### 3.3.1 二手市场
- 商品发布（图片、描述、价格、联系方式）
- 商品分类（书籍、电子产品、生活用品等）
- 商品搜索与筛选
- 商品收藏
- 在线咨询

#### 3.3.2 学习资源
- 学习资料分享（课件、笔记、题库）
- 资料分类（按学科、课程）
- 资料下载/预览
- 资料评价

#### 3.3.3 失物招领
- 失物发布
- 拾物发布
- 信息匹配推送
- 联系方式保护

### 3.4 活动模块

#### 3.4.1 活动发布
- 活动信息发布（时间、地点、人数限制）
- 活动分类（讲座、比赛、聚会等）
- 活动报名

#### 3.4.2 活动管理
- 报名人员管理
- 活动状态更新
- 活动提醒

### 3.5 搜索与发现

#### 3.5.1 搜索功能
- 全局搜索（用户、动态、资源）
- 关键词搜索
- 搜索历史

#### 3.5.2 推荐系统
- 个性化推荐
- 热门内容推荐
- 同校用户推荐

### 3.6 管理后台

#### 3.6.1 内容审核
- 动态审核
- 资源审核
- 用户举报处理

#### 3.6.2 数据统计
- 用户数据统计
- 内容数据统计
- 活跃度分析

#### 3.6.3 系统管理
- 用户管理
- 权限管理
- 系统配置

## 4. 非功能需求

### 4.1 性能需求
- 页面加载时间 < 2秒
- 支持并发用户数 > 1000
- 图片加载优化（压缩、懒加载）

### 4.2 安全需求
- 用户信息加密存储
- 敏感信息脱敏展示
- 防止恶意刷屏
- 内容安全审核

### 4.3 可用性需求
- 界面简洁友好
- 操作流程简单
- 提供操作引导
- 支持离线缓存

### 4.4 兼容性需求
- 支持微信小程序最新版本
- 兼容iOS和Android系统
- 适配不同屏幕尺寸

## 5. 技术架构

### 5.1 前端技术
- uni-app
- 组件化开发
- 状态管理

### 5.2 后端技术
- Node.js / nest
- RESTful API设计
- 数据库：PostgreSQL

### 5.3 第三方服务
- 微信开放平台
- 云开发服务
- 内容安全审核API
- 消息推送服务
